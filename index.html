Content Tabs

Content Tabs is a small & lightweight jQuery Plugin
for creating beautiful content tabs.

Description

Content Tabs is my first self-written jQuery Plugin. I created it for various reasons. One is that I want to get more proficient with JavaScript and jQuery in general. Another reason is that I like the idea of writing all of my code on my own. I would be really grateful for some feedback on the code quality.

Basic usage

In order for content tabs to work you need to setup your HTML like in the example below. Link the accordion.css and call the function contentTabs(); on the parent element of your tabs.

  1. Link CSS and JS File
  2. Setup your markup
  3. Assing function to parent element
Setup HTML 
          
$("#contentTabs").contentTabs();

Link CSS and JS File 
          
$("#contentTabs").contentTabs();

Call function on parent element 
          
$("#contentTabs").contentTabs();
